# Dev Tools

Local development tools for testing and debugging EMFN plugin functionality outside of WordPress.

## FIPS Risk Lookup Tool

**File:** `fips-risk-lookup.html`

A standalone HTML page that demonstrates the client-side risk logic from the EMFN Behavior Plugin without requiring WordPress, PHP, or any server-side processing.

### Features

- Simple FIPS code input form
- Direct filesystem lookups of NRI CSV data from the plugin's `assets/data/` directory
- Sample FIPS buttons for quick testing of various counties
- Displays FEMA NRI risk scores (≥50% threshold) for 18 hazard types
- Clean, responsive styling matching the plugin aesthetic

### Usage

1. **Open the HTML file in a browser:**
   ```bash
   # From the repository root
   open dev-tools/fips-risk-lookup.html
   # Or in your preferred browser
   ```

2. **Enter a 5-digit county FIPS code** in the input field and click "Lookup Risk Data"

3. **Or use the sample buttons** to quickly test with pre-populated FIPS codes:
   - 13121 - Fulton County, GA (Atlanta)
   - 06037 - Los Angeles County, CA
   - 48201 - Harris County, TX (Houston)
   - 12086 - Miami-Dade County, FL
   - 36061 - New York County, NY (Manhattan)
   - 17031 - Cook County, IL (Chicago)
   - 04013 - Maricopa County, AZ (Phoenix)
   - 53033 - King County, WA (Seattle)

### How It Works

1. **FIPS to State Mapping:** The tool extracts the state code from the first 2 digits of the FIPS code
2. **CSV Lookup:** Fetches the corresponding state CSV from `../plugins/emfn-behavior-plugin/assets/data/{ST}.csv`
3. **Data Parsing:** Parses the CSV and finds the row matching the entered FIPS code
4. **Risk Display:** Shows all hazards with NRI risk scores ≥50% (configurable threshold), sorted by severity

### Technical Details

- **No server required:** Works as a static HTML file with client-side JavaScript
- **Risk Threshold:** Set to 50% matching the plugin configuration
- **Hazard Types:** Supports all 18 FEMA NRI hazard types (Avalanche, Coastal Flooding, Cold Wave, Drought, Earthquake, Hail, Heat Wave, Hurricane, Ice Storm, Landslide, Lightning, Inland Flooding, Strong Wind, Tornado, Tsunami, Volcanic Activity, Wildfire, Winter Weather)
- **Data Source:** Uses the same per-state CSV files generated by `notebooks/US_disaster_risk_analysis.ipynb`

### Limitations

- Requires the CSV files in the plugin's `assets/data/` directory to be present
- Does not integrate with Google Places API or FCC API (direct FIPS input only)
- Does not save or persist lookup history
- Browser must support modern JavaScript (ES6+) and Fetch API

### Related Files

- Plugin JS: `plugins/emfn-behavior-plugin/assets/js/emfn-behavior-plugin.js`
- NRI Data: `plugins/emfn-behavior-plugin/assets/data/{ST}.csv` (51 state files)
- Data Generator: `notebooks/US_disaster_risk_analysis.ipynb`
