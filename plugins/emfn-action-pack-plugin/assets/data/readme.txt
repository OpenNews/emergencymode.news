EMFN Action Pack Plugin - Data Directory
============================================

Source: FEMA National Risk Index (NRI), December 2025 release
        https://hazards.fema.gov/nri/

Purpose
-------

This directory contains per-state county risk files consumed by
`fetchNriData()` in `assets/js/emfn-action-pack-plugin.js`.

Files in this directory
-----------------------

{ST}.csv (example: AL.csv)
  One file per U.S. state (plus DC), named by two-letter abbreviation.
  Generated by `notebooks/US_disaster_risk_analysis.ipynb`.

  Runtime usage:
    1) Browser geolocation/Places data resolves county + state abbreviation.
    2) Plugin fetches `${dataUrl}/{ST}.csv`.
    3) Plugin matches row by `county_fips` and builds risk output.

  Core columns:
    county_fips  5-digit county FIPS code (lookup key, zero-padded)
    state        Full state name (example: Alabama)
    county       County name (example: Autauga)

  Hazard score columns:
    {HAZARD}_risk_score for each NRI hazard code below.
    Values are percentile-like scores from 0 to 100.
    Blank value means not applicable or unavailable for that county/hazard.

NRI hazard codes used in these CSVs
-----------------------------------

AVLN  Avalanche           CFLD  Coastal Flooding     CWAV  Cold Wave
DRGT  Drought             ERQK  Earthquake           HAIL  Hail
HWAV  Heat Wave           HRCN  Hurricane            ISTM  Ice Storm
LNDS  Landslide           LTNG  Lightning            IFLD  Inland Flooding
SWND  Strong Wind         TRND  Tornado              TSUN  Tsunami
VLCN  Volcanic Activity   WFIR  Wildfire             WNTW  Winter Weather

Notes
-----

- The plugin filters hazards by `riskThreshold` in JS (default is 50).
- This folder currently stores generated state CSVs and this readme only.
