EMFN Behavior Plugin – Data Directory
======================================

Source: FEMA National Risk Index (NRI), December 2025 release
        https://hazards.fema.gov/nri/

Files
-----

{ST}.csv  (e.g. AL.csv)
  One file per U.S. state, named by two-letter abbreviation.
  Generated by notebooks/disaster_risk_analysis.ipynb from the full NRI dataset.
  Fetched at runtime by fetchNriData() in emfn-behavior-plugin.js using the
  state abbreviation resolved from the FCC Area API.

  Columns:
    county_fips   5-digit county FIPS code (e.g. 01001). Used as the lookup key.
    state         Full state name (e.g. "Alabama")
    county        County name (e.g. "Autauga")
    {PREFIX}_risk_score  (18 columns, one per NRI hazard — see NRI_HazardInfo.csv)
                  Composite NRI risk score for the county, 0–100.
                  Empty/blank = hazard is not applicable or no data for this county.

  The plugin filters to scores >= riskThreshold (default: 50) and presents the
  resulting hazards to the user, sorted by score descending.

NRI_HazardInfo.csv
  Reference table for the 18 NRI hazard types included in the per-state CSVs.
  Columns: OID_, Hazard (full name), Prefix (used to build column names in {ST}.csv),
           Service, Start, End, TotalYears, FrequencyModel,
           EAL_Building, EAL_Population, EAL_Agriculture, PeriodOfRecord, NRI_VER

  The 18 hazards and their column prefixes:
    AVLN  Avalanche           CFLD  Coastal Flooding     CWAV  Cold Wave
    DRGT  Drought             ERQK  Earthquake           HAIL  Hail
    HWAV  Heat Wave           HRCN  Hurricane            ISTM  Ice Storm
    LNDS  Landslide           LTNG  Lightning            IFLD  Inland Flooding
    SWND  Strong Wind         TRND  Tornado              TSUN  Tsunami
    VLCN  Volcanic Activity   WFIR  Wildfire             WNTW  Winter Weather

NRIDataDictionary.csv
  Full NRI data dictionary (~480 rows). Reference only — not loaded at runtime.
  Describes all fields in the upstream NRI dataset including those not present
  in the per-state CSVs (e.g. exposure, annualized loss, social vulnerability).
