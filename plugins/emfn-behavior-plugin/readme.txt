Behavior and customization plugin for emergencymode.news.

== Description ==

Custom CSS, JS and some data for EMFN needs.

== Installation ==

1. Upload the `emfn-behavior-plugin` folder to `wp-content/plugins/`.
2. Activate the plugin through the *Plugins* menu in WordPress Admin.
