Example scaffold for a Telex-generated plugin used on emergencymode.news.

== Description ==

This plugin is a starting-point scaffold intended to be replaced or extended
with functionality generated by Telex (https://telex.automattic.ai/) or
written by hand for the emergencymode.news Newspack site.

== Installation ==

1. Upload the `emfn-example-plugin` folder to `wp-content/plugins/`.
2. Activate the plugin through the *Plugins* menu in WordPress Admin.
